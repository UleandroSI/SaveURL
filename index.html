Salvar URL importantes para pesquisas. E te-las disponível e compartilhada online. Uma página de endereço único para salvar URLs importantes para serem visualizadas depois. Links para serem utilizados em projetos, ou trabalhos, ou TCC. A página é aberta para qualquer um que tenha o link, e pode ser compartilhada em grupos para contribuição.

Sobre o site

Modo de usar:
  • Crie uma página para armazenar os links.
  • Dê um nome para ser usado na criação do link.
  • Caso queira insira seu email para receber o link da página em seu email.
  • Obs.: O email é somente para envio do link, já que o link é o único meio de acesso, e não tem como recuperá-lo depois. Leia o Termo de Uso para mais informações sobre.
  • Ao criar à página poderá inserir links, que sempre que acessar novamente à página os links estarão disponíveis.
  • Distribua o link com quem desejá compartilhar os links salvos.
  • Lembrando que: Quem possuir o link de acesso conseguirá visualizar à página.

Caso goste do site e lhe seja útil, considere fazer uma doação para manter o projeto, que é mantido sem patrocínio.
